# Knowledge Distillery

See the [main README](../README.md) for the full project overview, topic file index, quick start guide, and architecture details.

This directory contains the 12 synthesized topic files, project docs (SOUL.md, DECISIONS.md, LEARNING-PATH.md, kickoff-guide.md), and the commands/ directory with slash command sources.

## For AI Agents

1. **Read [SOUL.md](SOUL.md) first.** It defines your relationship to this KB.
2. If `search_kb` is available, use it for targeted queries instead of reading entire files.
3. **Each file is self-contained.** Start with the one matching your current task.
4. **The two foundational files** are `prompt-engineering.md` and `context-engineering.md`.
5. **Follow cross-references only when needed.** Format: `(see [concept](file.md#section))`.
